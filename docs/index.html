mculed

Homebridge plugin to connect to nodeMCU devices via websockets

mculed(log: any, config: aliases, api: any)
Parameters
log (any)
config (aliases) Friendly names for your devices
api (any)
Example
config.json sample
{
   "platform": "mculed",
   "name": "mculed",
   "aliases": {
    "NODE-AC5812": "Kitchen Sink"
  }
}
